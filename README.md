# Pele Plena - Protocolo de Saúde Integrativa

## Sobre o Projeto
Landing page e plataforma informativa para o protocolo "Ciclo Facial Integrativo" da Plena Saúde.
Focado em medicina integrativa, dermatologia funcional e tecnologia avançada.

## Tecnologias

- **Framework**: React + Vite
- **Linguagem**: TypeScript
- **Estilização**: Tailwind CSS
- **UI Components**: Shadcn UI + Radix Primitives
- **Animações**: Framer Motion

## Como rodar localmente

1. Clone o repositório
2. Instale as dependências:
```bash
npm install
```
3. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

## Deploy

O projeto está pronto para deploy em plataformas como Vercel, Netlify ou GitHub Pages.
Certifique-se de configurar as regras de rewrite para Single Page Applications (SPA) se usar servidor estático.
